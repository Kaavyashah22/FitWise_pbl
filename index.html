FitWise

"An AI-powered fitness intelligence system that generates personalized diet and workout strategies using KNN classification and adaptive health metrics."

Problem Statement

Generic fitness apps provide static workout and diet plans that fail to adapt to individual body metrics such as BMI, TDEE, age group, and activity level. Users need intelligent, data-driven personalization.

Current Gap

No explainability, no confidence score, and no goal-specific optimization.

Our Solution

KNN-based classification engine that selects expert fitness clusters with explainable AI output and confidence scoring.

System Architecture

Frontend

React + TypeScript + Tailwind CSS Deployed on Vercel

AI Engine

Python Flask API KNN Classification (Goal-Specific Models) Confidence Score + Explainability Layer

Live Deployment

View Live Demo

System Impact

Personalization Accuracy 90%+

Using KNN classification and goal-specific expert models, FitWise dynamically selects optimal fitness strategies based on user health metrics.
